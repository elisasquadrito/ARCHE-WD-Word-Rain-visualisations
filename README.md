# ARCHE-WD Word Rain supplementary outputs

This repository contains supplementary outputs produced with Word Rain for the ARCHE-WD corpus and discussed in the submitted article.

The material is provided to allow closer inspection of the Word Rain visualisations and of the textual outputs generated by the tool. The Word Rain visualisations should be understood as exploratory outputs supporting the analysis. The interpretation presented in the article is further grounded in frequency data, Word Sketches, collocation measures and concordance analysis.

## Repository contents

### Main visualisation

* `outputs-ARCHE-WD-dotted4/ARCHE-WD-WordRain-output.pdf`

Combined PDF containing the aligned Word Rain visualisations for the ARCHE-WD corpus. The visualisations correspond to the four subcorpora analysed in the article:

1. Foresight Analysis
2. Vision and Mission
3. Key Messages and Preliminary Findings
4. Working Group Forms

### Word Rain plot-data JSON files

* `outputs-ARCHE-WD-dotted4/01_foresight_analysis-outputs-ARCHE-WD-dotted4.pdf.json`
* `outputs-ARCHE-WD-dotted4/02_vision_and_mission-outputs-ARCHE-WD-dotted4.pdf.json`
* `outputs-ARCHE-WD-dotted4/03_key_messages_and_preliminary_findings-outputs-ARCHE-WD-dotted4.pdf.json`
* `outputs-ARCHE-WD-dotted4/04_working_groups_forms-outputs-ARCHE-WD-dotted4.pdf.json`

The `.pdf.json` files are machine-readable plot-data files exported by Word Rain for each subcorpus. They are not raw word-vector files. Each file contains the data used to generate the corresponding Word Rain visualisation, including the list of plotted lexical items and n-grams, their score, projected x-axis position, plotted font size, raw frequency, relative frequency and display information.

In the exported JSON files included here, each entry in the `sorted_word_scores` field has the following structure:

`[word, score, x, fontsize, freq, relfreq, show]`

where `x` is the one-dimensional position used in the visualisation. The final `show` field contains display information for the plotted item as exported by Word Rain.

### Textual Word Rain output lists

Folder:

* `outputs-ARCHE-WD-dotted4/texts-outputs-ARCHE-WD-WordRain-output/`

Files:

* `01_Foresight_Analysis-outputs-ARCHE-WD-dotted4.txt`
* `02_Vision_and_Mission-outputs-ARCHE-WD-dotted4.txt`
* `03_Key_Messages_and_Preliminary_Findings-outputs-ARCHE-WD-dotted4.txt`
* `04_Working_Groups_Forms-outputs-ARCHE-WD-dotted4.txt`
* `all-outputs-ARCHE-WD-dotted4.txt`

The `.txt` files are tab-separated Word Rain output lists exported for each subcorpus and for the combined output. They list the lexical items and n-grams included in the Word Rain output, followed by numerical values generated by the tool, including score and frequency information. These files are provided as exported tool outputs; for the full structured plot data, see the corresponding `.pdf.json` files.

### New-word output lists

Folder:

* `outputs-ARCHE-WD-dotted4/texts-outputs-ARCHE-WD-WordRain-new-words/`

Files:

* `new_words_01_Foresight_Analysis-outputs-ARCHE-WD-dotted4.txt`
* `new_words_02_Vision_and_Mission-outputs-ARCHE-WD-dotted4.txt`
* `new_words_03_Key_Messages_and_Preliminary_Findings-outputs-ARCHE-WD-dotted4.txt`
* `new_words_04_Working_Groups_Forms-outputs-ARCHE-WD-dotted4.txt`

These files contain the lexical items and n-grams identified by the Word Rain `new words` function for each subcorpus. In the article, this function is used as an exploratory aid to identify items that appear only in specific parts of the aligned visualisation and may therefore point to emerging or under-represented themes.

## Scope of the repository

This repository includes Word Rain visual outputs and tool-generated output lists only. It does not include the raw ARCHE-WD corpus texts.

The files are provided as supplementary material for inspection and transparency. They do not replace the close-reading analysis presented in the article.
